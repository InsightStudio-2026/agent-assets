# 行为准则 · Code of Conduct

## 📜 我们的承诺

为了营造一个开放友善的环境，作为贡献者和维护者，我们承诺：参与本项目的每个人都不受骚扰，无论年龄、体型、可见或不可见的残疾、族裔、性别认同与表达、经验水平、教育、社会经济地位、国籍、个人外貌、种族、宗教或性取向身份。

我们承诺以促进开放、包容、多元化、健康社区的方式行动和互动。

## 📑 我们的准则

创造积极环境的行为示例：

- 对他人展现同理心和善意
- 尊重不同意见、观点和经验
- 给予建设性反馈，也优雅地接受反馈
- 承担责任并向受我们错误影响的人道歉
- 关注的不仅是个人，更是整个社区的福祉

不可接受的行为示例：

- 性化语言或图像、性关注或挑逗
- 钓鱼、辱骂或贬损的评论；人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人私人信息（如物理或电子地址）
- 其他在专业环境下可被合理认定为不当的行为

## 🏛️ 职责范围

本行为准则适用于所有项目空间（代码仓库、Issue、PR、Discussions、社交媒体、公共空间），并在个人在公共空间正式代表项目时同样适用。

## 📣 执行

辱骂、骚扰或其他不可接受的行为可通过私下联系仓库维护者举报（联系方式见仓库 Insights → Contributors）。所有投诉都会被及时、公正地审查和处理。

维护者有权利和义务删除、编辑、拒绝不符合本行为准则的评论、提交、代码、wiki 编辑、Issue 和其他贡献。
